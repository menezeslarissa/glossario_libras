Importante: para que a aplicação funcione corretamente devem ser instaladas as versões dos softwares indicadas abaixo:

PYTHON

Python (64 bits): https://www.python.org/ftp/python/3.7.0/python-3.7.0-amd64.exe

Python (32 bits): https://www.python.org/ftp/python/3.7.0/python-3.7.0.exe

MYSQL (5.5 ou superior)

Windows (32-bit), MSI Installer 8.0.13, 313.8M https://dev.mysql.com/downloads/windows/installer/8.0.html

Em seguida, abra o prompt de comando e execute os seguintes comandos:

pip3 install opencv-python

pip3 install zipfile36

pip3 install pymysql

pip3 install pygame

pip3 install pyqt5
