Conectando o Glossário de Libras com MySQL

Para fazer a conexão do Glossário de Libras com o MySQL deve-se abrir o sistema.

Após isso, deve-se criar uma conexão.

Posteriormente, surgirá uma tela para cadastrar o nome da conexão. Em seguida deve-se clicar no botão 'ok'.

Logo após, aparecerá a conexão cadastrada e deve-se clicá-la.

Seguindo, abrirá outra tela deve-se escrever 'root' e clicar em 'ok'.

A seguir, deve-se abrir o arquivo 'librasbd.sql'. Que se encontra na pasta onde foi salvo os arquivos do Glossário de Libras ou poderá encontrá-lo neste link:

Após aberto o arquivo execute o script.

Em seguida, aparecerá na parte inferior o script executado.
